# 🏛️ Abdullah Al Masud - Professional Islamic Profile & CV

A premium, responsive, and aesthetically pleasing personal profile web application for **Abdullah Al Masud**. This project showcases a blend of Islamic scholarship, academic achievements, and personal values through a modern "Emerald & Gold" themed UI.

## ✨ Features

*   **Premium Visuals:** A sophisticated color palette (`#051413` Deep Green and `#d9b48b` Gold Sand) reflecting a professional Islamic identity.
*   **Dynamic Role Switcher:** JavaScript-powered typing effect showcasing various titles (Da'ee, Researcher, Student).
*   **Comprehensive CV:** Sections for Institutional Education, Religious Training, Family Background, and Personal Ideals.
*   **Fully Responsive:** Optimized for all devices—from mobile screens to large desktop monitors.
*   **Interactive UI:** Hover effects, smooth scroll animations using Intersection Observer, and glassmorphism navigation.

## 🛠️ Built With

- **HTML5** & **Semantic Tags**
- **CSS3** (Custom Properties, Flexbox, Grid, and Animations)
- **JavaScript** (Typing Effects and Scroll Animations)
- **Google Fonts** (Playfair Display for headings & Inter for body text)
- **FontAwesome** (Icons)
